# Web Design, IT Support, and Content Writing Portfolio

A complete beginner-friendly portfolio project designed for freelance presentation and GitHub publishing. This repository demonstrates front-end web design, IT support documentation, and SEO content writing in one organized structure.

## Features

- Responsive, modern portfolio website
- Structured sections: Home, About, Services, Portfolio Projects, and Contact
- Interactive JavaScript features:
  - Mobile navigation toggle
  - Contact form validation
- IT support documentation for troubleshooting and best practices
- SEO-friendly blog writing sample

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)
- Markdown

## Project Structure

```text
portfolio-project/
├── index.html
├── style.css
├── script.js
├── README.md
├── content/
│   └── blog_sample.md
└── docs/
    └── IT_support_guide.md
```

## How to Run the Project

1. Clone or download this repository.
2. Open the `portfolio-project` folder.
3. Open `index.html` in your web browser.

Optional local server command:

```bash
cd portfolio-project
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.
